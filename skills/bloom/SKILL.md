# Bloom

## The world model

Bloom owns the brand layer — the brand's visual DNA and a brand-aware system prompt are applied automatically, so your prompt only supplies the **subject**, **composition**, and **medium**. [`rules/prompting.md`](rules/prompting.md) is the full version.

## Required reading

For any Bloom generation, edit, resize, or brand-image task, read [`rules/workflow.md`](rules/workflow.md) before calling tools. It covers checking for references first.

Then read [`rules/prompting.md`](rules/prompting.md) for writing the prompt text.
